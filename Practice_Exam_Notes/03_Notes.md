# Practice Exam 03 Notes

## AWS Config

AWS Config is a service that enables us to assess, audit, and evaluate the configurations of our AWS resources.

AWS resources are entities we can work with such as EC2 instances, CloudFormation stacks, or S3 buckets.

## AWS Storage Gateway

AWS Storage Gateway is a hybrid cloud storage service that connects our existing on-premise environments with the AWS Cloud.

The three different gateway types support by AWS Storage Gateway are: Tape Gateway, File Gateway, and Volume Gateway.

## AWS Compute Optimizer

AWS Compute Optimizer helps us identify the optimal AWS resource configurations for Amazon EBS volumes, AWS Lambda functions, Amazon EC2 instances, and Amazon EC2 Auto Scaling groups.

## Amazon Athena

Amazon Athena is an interactive query service that makes it easy to analyze data in Amazon S3 using standard SQL.
